HTML

  • html lang=en - the page is written in english
  • head - contains the metadata for our web application
  • title - defines the title for the webpage
  • style - defines the CSS styles associated with a website
  • base - defines the base URL for a webpage
  • link - connects an exdternal resource to the HTML document
  • meta - defines such things like character set, descriptions, keywords, author, and viewport
  • script - embeds client-side scripts in the HTML document

Git

  • pwd - print working directory
  • ls - shows a list of working files in your directory
  • cd - change directory
  • 'cd ..' - go back one level
  • mkdir - make directory
  • touch - creates a file inside a directory (touch first-day.txt)
  • open - open the file (open first-day.txt)
  • Git must precede every statement
  • -b - creates a new branch (git checkout -b feature/starter-code)
  • add - adds modifications in the current working branch to the staging
  • git pull origin main - pull - to receive a branch's modifications into the local environment
  • origin - indicate the source of the pull will be in the GitHub repo
  • Main - indicates the branch
